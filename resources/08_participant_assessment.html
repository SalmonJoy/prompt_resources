Section C · Practical scenario

Fictional scenario
Your team has been asked to prepare a short internal note announcing that a fictional internal request form will be piloted for two weeks. You have a few rough facts: the pilot dates, which team is involved, and that feedback will be gathered at the end. Some details are still unconfirmed. The note will go to all staff in the affected team.

Using the framework, complete the following. This is a fictional workshop scenario and not an official record.

  1. Define the business goal
  2. Identify the audience
  3. List the source information you would use
  4. Define the constraints
  5. Specify the output format
  6. Write a complete structured prompt
  7. Write one critique prompt
  8. Write one refinement prompt
  9. Explain how the final output should be validated
Responsible use
Use fictional information only. Remember that AI output is a draft until a human reviews it, and that internal company policy takes precedence over any workshop guidance.
