Advanced AI Prompt Framework

Frame the Problem → Ground the Model → Engineer the Result

Before Prompting

  • What is my real objective?
  • Who is the output for?
  • What information is available?
  • What must ChatGPT not assume?
  • What does a successful output look like?

Build the Prompt

  • Goal
  • Role
  • Context
  • Source
  • Audience
  • Instructions
  • Constraints
  • Output format
  • Quality criteria

After Receiving the Output

  • Check instruction compliance
  • Check factual accuracy
  • Identify unsupported claims
  • Identify missing information
  • Refine the response
  • Obtain human review
  • Verify before use
Responsible use
Never upload confidential, personal, proprietary, audit-sensitive, clinical, formula-related, regulatory or credential information into public AI tools without organisational approval.
