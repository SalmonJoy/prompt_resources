Print the complete participant pack

Browsers print one page at a time, so the most reliable way to print the whole pack is to print each resource individually using its own Print this resource button. This keeps every A4 layout, page break and one-page reference correct.

  1. Open each resource using its Open button above.
  2. On the resource, click Print this resource (top of the page).
  3. In the print dialog, choose A4 paper and enable Background graphics for full colour.
  4. To keep a digital copy, choose Save as PDF as the printer destination.

Tip: Resources 02, 03 and 04 are designed as single-page workstation references — keep them at 100% scale so they stay on one page.

The button above opens each resource in its own tab with the print dialog ready. If your browser blocks pop-ups, allow them for this page or print each resource individually.

Instructions for participants

  • Start with the Participant Handbook (01) for the full method, then keep the poster (02) and checklists (03, 04) at your workstation.
  • Use the Prompt Library (05) as a daily starting point — always replace the variables in square brackets before running a prompt.
  • Practise with the Case-Study Workbook (06) using fictional examples only.
  • Every prompt output is a draft until a human reviews it. Verify facts, references and calculations before use.
  • Do not enter confidential, personal, proprietary, audit-sensitive, clinical, formula-related, regulatory or credential information into public AI tools without organisational approval.
  • Your organisation’s approved information-security, data-privacy, quality, regulatory and AI-usage policies always take precedence over this workshop material.
