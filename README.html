How to use this resource pack

This guide explains how to open, navigate and print the participant resources for the workshop “Advanced Prompt Engineering with ChatGPT and Gamma AI”.

Open the pack (index.html)

How to open the resource pack

  • Keep the whole folder together. All files reference each other using relative paths.
  • Double-click index.html to open the pack in your web browser. No internet connection is required — everything works offline.
  • If you received a ZIP file, extract (unzip) it fully first, then open index.html from the extracted folder. Do not open files from inside the compressed ZIP.

How to use index.html

  • index.html is the home page. It lists all nine resources with a short description.
  • Use the Open button on any resource to read it on screen.
  • Use the Print button to open a resource with the print dialog ready.
  • The Print Complete Participant Pack section explains how to print everything reliably.

How to print individual resources

  1. Open the resource you want to print.
  2. Click Print this resource at the top of the page (this button is hidden on the printout).
  3. In the print dialog, set the paper size to A4 and the scale to 100%.
  4. Print, or choose Save as PDF to keep a digital copy.

Recommended browser settings

  • Use a current version of a mainstream browser (for example Microsoft Edge, Google Chrome or Firefox).
  • Set paper size to A4 and orientation to Portrait (the pack sets this automatically where supported).
  • Set margins to Default. The pack already includes suitable print margins.
  • Keep scale at 100%, especially for the one-page references, so they stay on one page.

How to enable background graphics when printing

Colour panels and headings print best with background graphics turned on. The resources are still fully readable in black and white if it is off.

  • Edge / Chrome: In the print dialog, open More settings and tick Background graphics.
  • Firefox: In the print dialog, open More settings and tick Print backgrounds.

How to print to PDF through the browser

  1. Open the resource and click Print this resource.
  2. In the print dialog, change the Destination or Printer to Save as PDF (or Microsoft Print to PDF).
  3. Confirm A4 and 100% scale, then save.

One-page workstation references

These resources are designed to fit on a single A4 page so you can pin them at your desk. Keep the print scale at 100% so they stay on one page:

  • Resource 02 — Advanced Prompt Framework Poster
  • Resource 03 — Prompt Review Checklist
  • Resource 04 — Responsible AI Checklist
